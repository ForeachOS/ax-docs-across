[[across-test]]
= IV. Across Test

Across Test is a utility library that provides helpers for integration testing of Across modules in both a standard and web context.
Across Test builds on top of the Spring testing library.

[discrete]
== Artifact
[source,xml,indent=0]
[subs="verbatim,quotes,attributes"]
----
	<dependencies>
		<dependency>
			<groupId>com.foreach.across</groupId>
			<artifactId>across-test</artifactId>
			<version>{across-version}</version>
		</dependency>
	</dependencies>
----
